## 列表

### 无序列表

- 项目1
- 项目2

  - 子项目2

- 项目3

* 项目1

* 项目2
  
  * 子项目1

1. 第一项
2. 第二项
   1. 子项目1
   2. 子项目2

- [x] 已完成任务
- [ ] 未完成任务
- [x] ~~已完成并划掉的任务~~
- [ ] 另一个未完成任务

### 代码块

使用三个反引号：

```
这是一个普通代码块
无语法高亮
```

带语法高亮的代码块：

```javascript
function helloWorld() {
    console.log("Hello, World!");
    return "Hello, World!";
}

const message = helloWorld();
```

```python
def hello_world():
    print("Hello, World!")
    return "Hello, World!"

message = hello_world()
```

```html
<!DOCTYPE html>
<html>
<head>
    <title>页面标题</title>
</head>
<body>
    <h1>这是一个标题</h1>
    <p>这是一个段落。</p>
</body>
</html>
```

```css
.container {
    width: 100%;
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

h1 {
    color: #333;
    font-size: 2em;
}
```

## 学习建议

1. **从基础开始**：先掌握标题、段落、列表等基本语法
2. **多练习**：在支持Markdown的编辑器中实际练习
3. **查看渲染效果**：对比源码和渲染后的效果
4. **了解差异**：不同的Markdown解析器可能支持的语法略有不同
5. **工具推荐**：
   - 编辑器：Typora、Mark Text、VS Code
   - 在线编辑器：Dillinger、StackEdit
   - 平台：GitHub、GitLab、掘金、简书等

## 常用快捷语法速记

```
标题：# ## ### #### ##### ######
粗体：**文本** 或 __文本__
斜体：*文本* 或 _文本_
代码：`代码` 或 ```代码块```
链接：[文本](URL)
图片：![alt](URL)
引用：> 文本
列表：- 或 * 或 1.
表格：| 列1 | 列2 |
分割线：--- 或 *** 或 ___
删除线：~~文本~~
```

记住，熟能生巧！多练习这些语法，很快就能熟练掌握Markdown了。
